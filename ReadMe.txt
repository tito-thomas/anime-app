The supporting zip folder, "anime-app.zip" contains the following:

- Full code for the recommendation engine: preprocessing.ipynb, embedding.ipynb, 
neighbours.ipynb
Original dataset used can be found at: https://www.kaggle.com/datasets/azathoth42/myanimelist?select=anime_cleaned.csv

- Full web application including models, views, css, 
controllers, scripts, web configuration files. This is housed in "anime-site"
Anime cover images have been removed due to file size constraints.

- SQL queries for creating the "UserDetails" and "AnimeDetails" tables: 
"Queries" folder

- Code for image scraper used to collect anime cover images: "scrape" folder
